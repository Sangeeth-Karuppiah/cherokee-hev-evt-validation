# Reporting

This folder contains templates for EVT status reporting and risk tracking.

## Contents

- **weekly_status_template.md** - Weekly EVT status report structure (coming soon)
- **risk_register_template.xlsx** - Active issue tracking and escalation matrix (coming soon)

## Purpose

These reporting templates enable consistent communication of EVT progress, risks, and issues to program management and cross-functional stakeholders.

## Report Frequency

- **Weekly Status**: Every Monday, covering previous week's testing and upcoming week's plan
- **Risk Register**: Updated daily, reviewed in weekly risk review meetings

---

*Note: All templates are anonymized and do not contain confidential program information.*
