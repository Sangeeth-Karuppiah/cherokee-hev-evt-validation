# Tools

This folder contains Python scripts and Jupyter notebooks for test optimization and data analysis.

## Contents

- **dyno_cycle_optimizer.ipynb** - Python notebook for optimizing test sequence to minimize dyno downtime (coming soon)
- **test_coverage_checker.py** - Script to validate DVP&R coverage against requirements (coming soon)
- **synthetic_data_generator.py** - Generate dummy test data for demonstration (coming soon)

## Purpose

These tools automate test planning calculations, check coverage completeness, and generate synthetic data for portfolio demonstration.

## Requirements

```
python >= 3.8
pandas
matplotlib
openpyxl
```
